# Delete Jobs Script

A Python script for managing and deleting job entries.

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Copy `.env.example` to `.env` and configure your environment variables

## Usage

Run the script:

```bash
python main.py
```
